# BillBuddy AI

An intelligent bill management and analysis application powered by Google's Gemini AI. Track your bills, analyze spending patterns, and get AI-driven insights with an intuitive, modern interface.

## 🌐 Live Demo

**Try the app now:** [https://billbuddy-ai-97323745133.asia-southeast1.run.app/](https://billbuddy-ai-97323745133.asia-southeast1.run.app/)

## ✨ Features

- **AI-Powered Bill Analysis** - Get intelligent insights about your bills using Gemini AI
- **Bill Tracking** - Record and manage your bills with date and amount tracking
- **Spending Analytics** - Visualize your spending patterns with interactive charts
- **Image Support** - Upload bill images for analysis
- **Dark Mode** - Eye-friendly dark theme option
- **Responsive Design** - Works seamlessly on desktop and mobile devices
- **Real-time Chat** - Interactive AI chat for bill-related questions and advice
